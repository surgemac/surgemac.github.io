什么是Shadowrocket？

Shadowrocket是一款专为iOS平台开发的网络代理软件，旨在帮助用户突破网络封锁和访问地域受限的内容。它允许用户通过多种网络协议（如Shadowsocks、Vmess、HTTP和HTTPS等）进行代理配置，从而绕过防火墙，保护用户的隐私并提升上网自由度。与其他代理工具相比，Shadowrocket不仅操作简便，而且具备强大的流量监控、规则管理和捷径功能，让用户能够更加灵活地管理网络连接。

Shadowrocket的核心功能

1. 网络代理功能

Shadowrocket支持多种网络协议，包括Shadowsocks、Vmess、HTTP和HTTPS协议。这些协议在安全性和隐蔽性方面有不同的优势，用户可以根据自己的需求选择最合适的协议。通过使用这些协议，用户可以通过代理服务器隐藏自己的真实IP，保护隐私，同时突破网络封锁，访问全球互联网资源。

此外，Shadowrocket还允许用户为不同的网络环境配置代理设置。比如，用户可以在Wi-Fi和移动数据网络下分别设置不同的代理节点，以保证最佳的网络体验。

2. 流量监测功能

Shadowrocket提供了实时的流量监控功能，用户可以详细查看每个应用程序的流量消耗。这一功能对于需要频繁使用网络的用户尤其有帮助，能够帮助用户了解哪些应用占用了大量带宽，从而优化网络资源的分配。用户还可以设置流量限制，避免因网络使用过多而导致的额外费用或带宽拥堵。

3. 规则管理

Shadowrocket具备强大的规则管理功能，用户可以自定义网络规则，以实现更精细的流量管理。例如，用户可以设置常用的网站使用直连方式（不通过代理），而对特定的被封锁网站进行代理访问。除此之外，Shadowrocket还内置了多个规则库，用户可以根据需求选择合适的规则进行使用，从而优化网络体验。

4. 捷径功能

Shadowrocket与iOS的捷径功能无缝结合，允许用户通过一键切换，快速根据需求选择不同的代理。通过捷径功能，用户无需每次都手动更改设置，简单的快捷操作就可以在不同的网络环境中轻松切换代理配置。这使得用户在日常使用中更加便捷，极大提升了使用效率。

FAQ

Nekobox 和 Nekoray 有何关系？
Nekobox 是 Nekoray 的移动端衍生版，界面简洁，主要适配 Android 平台。两者使用同样的核心逻辑与配置结构，因此订阅文件和节点信息可以兼容共享。
Surfboard 的策略组可以根据延迟自动切换节点吗？
支持。Surfboard 可定期测速节点延迟，并结合策略组自动选择延迟最低的节点。这样用户无需手动切换节点即可保持高速访问，提高网络体验和稳定性。
Netch 全局与规则代理如何合理选择？
全局代理适合游戏或全局科学上网，将所有流量通过节点；规则代理仅对指定应用或 IP 走代理，可提高带宽利用率和访问效率，根据使用场景灵活选择模式。
Mellow 的 TUN 模式与 SOCKS5 代理区别是什么？
TUN 模式拦截系统 IP 层流量，实现全局代理；SOCKS5 代理通常只影响特定应用。TUN 更适合全局加速、游戏和视频应用，保证所有流量走节点，提高网络稳定性和速度。
VLESS + XTLS 的传输效率比 Trojan 高吗？
在高延迟环境下，VLESS + XTLS 的性能通常比 Trojan 高。XTLS 去掉了冗余加密开销，降低 CPU 占用，提高多连接场景下的吞吐率，尤其在服务器高负载情况下表现更佳。
Clash 的 Profile URL 是否可以同时添加多个？
可以。Clash 支持导入多个 Profile URL，用户可在“配置”面板切换不同订阅源。多个配置之间互不干扰，方便区分不同地区或用途的线路。
如何合理设置 Clash 的 proxy-group 实现自动切换？
创建包含多个节点的策略组（如 `url-test` 或 `fallback`），设置合适的检测 URL 与检测频率，启用延迟优先或可用性优先策略。这样客户端会自动选择延迟最低或可用节点，提升稳定性。
V2RayN 与 WinXray 哪个更易用？
V2RayN 界面更简洁，功能更稳定，而 WinXray 提供了更现代的界面和自动测速功能。对于初学者来说，V2RayN 上手更快；而追求高级功能的用户可能更喜欢 WinXray。
VMess 使用 AEAD 加密的优势是什么？
AEAD 加密提供更强的安全性和抗重放攻击能力，同时隐藏流量特征，降低被 DPI 检测风险。在高封锁环境下使用，可提升连接稳定性和长期可用性。
SSR 的 UDP 转发功能如何启用？
SSR 支持 UDP 转发，可通过客户端设置开启。启用后，游戏和语音等 UDP 流量可通过节点代理，降低丢包率和延迟，提高实时网络应用体验。
