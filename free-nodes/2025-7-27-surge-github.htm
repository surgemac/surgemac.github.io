五、节点添加与配置指南（以 vmess 为例）

  1. 打开 Shadowrocket，点击右上角“+”号；

  2. 选择协议类型为 vmess；

  3. 输入如下关键信息：

    • 地址（host）

    • 端口（port）

    • UUID（用户身份认证）

    • 加密方式（默认选择 auto 即可）

    • 传输方式（WebSocket / TCP / HTTP）

    • TLS 设置（若支持 HTTPS）

    • 伪装域名（若使用 CDN 转发）

  4. 保存后，返回首页，开启连接；

  5. 成功连接后状态变为绿色，网络通畅。

Shadowrocket 本身是合法工具，但其用途决定法律合规性。建议用户仅用于信息获取、技术交流和隐私防护等正当用途，勿涉违法活动。
